Many organisations in South Africa still rely on manual, email-based, or spreadsheet-driven leave approval processes, which result in:

Delayed approvals

No audit trail

Poor accountability

Security risks

Lack of visibility for employees, managers, and HR

This project solves that by providing a secure, role-based, multi-stage leave approval system that mirrors how real enterprises operate.

The system allows employees to submit leave requests that move through a controlled workflow:

Employee submits a leave request

Manager reviews and approves/rejects

HR performs final approval or rejection

All actions are logged for auditing

The application is built as a real-world internal business system
